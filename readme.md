# Requirements

1. Node and npm
2. bower: `npm i bower -g`

# Set up

Set up dependencies:

1. `npm i` in the root.
2. `npm i` in the server folder
3. `bower i` in the client folder
3. `npm i nodemon -g`

# Start Development

After installing the dependencies, run the following to start development:

`npm start`

Then go to `http://localhost:8051` to see the app.
